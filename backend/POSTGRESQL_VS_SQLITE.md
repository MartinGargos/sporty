# PostgreSQL vs SQLite - Který použít?

## PostgreSQL (doporučeno pro produkci)

### ✅ Výhody:
- Výkonnější pro větší aplikace
- Lepší podpora pro více uživatelů současně
- Pokročilejší SQL funkce
- Lepší pro produkci

### ❌ Nevýhody:
- Vyžaduje instalaci
- Více konfigurace
- Těžší na setup

## SQLite (jednodušší pro vývoj)

### ✅ Výhody:
- **Žádná instalace** - funguje okamžitě
- Jednoduchý setup
- Stačí jedna soubory databáze
- Ideální pro vývoj a testování

### ❌ Nevýhody:
- Omezenější pro větší aplikace
- Horší pro více současných uživatelů
- Pro produkci by bylo lepší PostgreSQL

## Doporučení

- **Pro teď (vývoj)**: Můžeme použít SQLite - je to rychlejší na setup
- **Pro produkci**: Později migrujeme na PostgreSQL

**Chceš použít SQLite pro jednodušší start?** Řekni mi a upravím backend pro SQLite.
